Tarmola Code Garage

Esittely

Olen ammattitaitoinen ohjelmistokehittäjä, Tieto- ja viestintätekniikan kouluttaja, perheenisä ja yrittäjä Porvoosta. Kokemus kattaa Linux-, Android, ja Windows-ympäristöjen ylläpidon ja asennukset, Python-, JavaScript- ja C#-kehityksen kuin verkkosivut sekä verkkopalveluiden PWA-toiminnallisuudet. Hyödynnän työssäni myös kehittyneimpiä tekoälytyökaluja, mutta perusosaaminen on rakennettu perinteisen kovan työn kautta.

Osaamisalueet

  • Windows, Android, Linux -asennukset, asetukset sekä käytön opastus
  • Ohjelmistokehitys (HTML, CSS, Bootstrap, Git, Python, PowerShell, C#, .NET MAUI, Xamarin, Node.js, React.js, Django, ASP.NET, Codex ja JavaScript)
  • Verkkosivujen sisällön suunnittelu, toteutus, hostaus, hakukoneoptimointi ja PWA-ominaisuudet
