Tietosuojaseloste

Tällä sivulla kerrotaan selkeästi, mitä henkilötietoja Tarmola Code Garage voi käsitellä asiakastyön yhteydessä, mihin tarkoitukseen tietoja käytetään ja missä tilanteissa tietoja voi näkyä palveluntarjoajille.

Mitä tietoja voidaan käsitellä

Tarmola Code Garage ei ylläpidä erillistä asiakasrekisteriä omissa järjestelmissään.

Asiakastyön yhteydessä voidaan kuitenkin käsitellä tarpeen mukaan esimerkiksi seuraavia tietoja:

  • nimi
  • puhelinnumero
  • osoite
  • sähköpostiosoite
  • laskutuksen yhteydessä tarvittavat tiedot
  • asiakkaan laitteella näkyvät tiedot vain siinä laajuudessa kuin sovittu tukityö sitä edellyttää

Jos asiakas soittaa kotikäynnistä, hän voi kertoa nimensä ja osoitteensa käynnin sopimista varten.

Jos asiakas maksaa MobilePaylla, maksajan nimi ja puhelinnumero voivat näkyä MobilePay-järjestelmässä maksutapahtuman yhteydessä.

Jos asiakas haluaa laskun, laskutusta varten voidaan käsitellä nimeä, puhelinnumeroa, osoitetta ja tarvittaessa sähköpostiosoitetta.

Mihin tarkoitukseen tietoja käsitellään

Tietoja voidaan käsitellä vain asiakastyön hoitamiseen, yhteydenpitoon ja tarvittaessa laskutukseen.

Tietoja ei kerätä markkinointirekisteriksi.

Tietojen lähteet

Tiedot saadaan pääasiassa asiakkaalta itseltään esimerkiksi puhelun, viestin, sähköpostin, kotikäynnin sopimisen tai laskutuksen yhteydessä.

Maksutapahtuman yhteydessä tietoja voi näkyä MobilePay-palvelussa.

Laskutuksen yhteydessä tietoja voi muodostua OP Kevytyrittäjä -palveluun.

Tietojen luovutus ja palveluntarjoajat

Tietoja ei luovuteta tarpeettomasti ulkopuolisille.

Tietoja voidaan käsitellä seuraavien palvelujen yhteydessä siltä osin kuin palvelun käyttö sitä edellyttää:

  • MobilePay maksutapahtumissa
  • OP Kevytyrittäjä laskutuksessa

Tietojen säilytys

Tarmola Code Garage ei säilytä asiakastietoja omissa erillisissä muistiinpanoissa asiakasrekisterinä.

Käteis- ja MobilePay-maksuissa asiakkaiden nimiä ei kirjata kirjanpitoon.

Jos laskutus tehdään, laskutukseen liittyviä tietoja voi löytyä OP Kevytyrittäjä -palvelun laskutushistoriasta. Tietoja säilyy tällöin siinä laajuudessa ja sen ajan kuin kyseinen palvelu ja laskutuksen hoitaminen edellyttävät.

Rekisteröidyn oikeudet

Rekisteröidyllä on soveltuvan tietosuojalainsäädännön mukaisesti oikeus saada tieto siitä, käsitelläänkö häntä koskevia tietoja, pyytää pääsyä omiin tietoihinsa sekä pyytää virheellisen tiedon oikaisemista.

Rekisteröity voi myös pyytää tietojen poistamista tai käsittelyn rajoittamista, jos siihen on lain mukainen peruste.

Lisäksi rekisteröidyllä on oikeus tehdä valitus tietosuojavaltuutetulle.

Koska Tarmola Code Garage ei ylläpidä erillistä asiakasrekisteriä, oikeuksien toteuttaminen riippuu käytännössä siitä, mitä tietoja kulloinkin on käsitelty esimerkiksi laskutuksen yhteydessä.

Asiakkaan laitteella työskentely

Asiakkaan laitteella työskentelyn tarkoitus on tehdä vain sovittu tukityö.

Asiakkaan tietoja käsitellään vain työn suorittamisen kannalta tarpeellisessa laajuudessa. Tarkoitus ei ole käydä asiakkaan tiedostoja, viestejä tai muita henkilökohtaisia tietoja läpi muuten kuin silloin, jos se on välttämätöntä sovitun ongelman ratkaisemiseksi.

Yhteydenotto tietosuoja-asioissa

Tietosuojaan liittyvissä asioissa voi ottaa yhteyttä Tarmola Code Garageen.
